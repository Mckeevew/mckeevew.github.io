MIPS Programming on Basys3

I have worked with Verilog (hardwar description language) code for a MIPS processor. The Code ran on a Basys 3 FPGA. I extended the processor code to handle LCD, push buttons, audio buzzer, ect. Besides extending the processor, I also wrote MIPS Assembly and C code to make use of all of theseh modifications. The code is not able to be public posted in Git.
